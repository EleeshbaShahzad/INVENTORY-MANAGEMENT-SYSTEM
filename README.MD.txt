Inventory Management Lookup System

 Overview
A comprehensive sales analytics dashboard for tracking electronics product performance. Built with Excel, PivotTables, and Power Query.

 Key Features
- Data cleaning :Blank ,Spelling error, format error by using power query 
- KPI Cards: Total Sales, Top Category, Avg Price
- Charts: Bar Chart (Sales by Product), Donut Chart (Category Share)
- Slicers: Filter by Category, Product, Price Range
- Insights: Monitor category drives 19% revenue ($5.6B)

 Tech Stack
- Excel
- Power Query
- PivotTables
- VBA (optional)

 File Structure
- data: Raw sales data
- Inventory data :Cleaned Data
- dashboard :Interactive dashboard file
- report: Analysis report (PDF/Word)

 Insights
- Monitors contribute 19% sales ($5.6B)
- Storage + Networking = 21% combined share
- Average Price: $7,407 (Premium market)
